# Defect Decision Guide for Current STS2 Data

This guide is based on `runtime/reference-library.json`, not STS1 memory. It focuses on current STS2 Defect decisions: what to draft, what to cut, what to upgrade, and where the event set is especially favorable or dangerous.

## What Is Different From STS1

- The class has more than one real engine now. Standard orb/focus is still strong, but there is also a real zero-cost shell and a genuine status-matter package.
- `Glass` is a new orb family in the current pool, so cards like `Glasswork`, `Refract`, and `Spinner` matter.
- Status creation is not pure downside. `Iteration`, `Smokestack`, `Trash to Treasure`, `Flak Cannon`, and `Rocket Punch` all reward you for creating or consuming statuses.
- `Power Cell` is a major identity relic here because it makes the zero-cost package much more consistent than old Defect.

## Default Draft Plan

If no archetype is open yet, draft in this order:

1. Reliable orb generation.
2. Focus and card draw.
3. Frontloaded block.
4. One scaling package.
5. Removes and transforms.

In practice, that means early rewards should usually prefer cards like `Ball Lightning`, `Cold Snap`, `Coolheaded`, `Glacier`, `Defragment`, `Charge Battery`, `Skim`, `Sweeping Beam`, `Beam Cell`, and `Loop` over slow payoff cards with no orb support.

## Best Packages

### 1. Orb Focus Core

This is still the most reliable default Defect plan.

Core enablers:

- `Ball Lightning`
- `Cold Snap`
- `Coolheaded`
- `Glacier`
- `Charge Battery`

Best payoffs:

- `Defragment`
- `Loop`
- `Gold-Plated Cables`
- `Data Disk`
- `Runic Capacitor`
- `Coolant`
- `Hailstorm`

Selection rule:

- `Defragment` and `Data Disk` are major power spikes. Take them aggressively.
- `Runic Capacitor` is best when your deck is already channeling often.
- `Coolant` and `Hailstorm` improve sharply once the orb spread is stable.

### 2. Zero-Cost Recycling

This is one of the clearest STS2 Defect upgrades over old Defect.

Core enablers:

- `Claw`
- `Beam Cell`
- `FTL`
- `Hotfix`
- `TURBO`

Best payoffs:

- `All for One`
- `Scrape`
- `Feral`
- `Power Cell`
- `Tesla Coil`
- `Momentum Strike`

Selection rule:

- Do not take `All for One` before you have enough good zero-cost cards to return.
- `Power Cell` is one of the strongest class relics for this shell because it increases early hit density every combat.
- `Scrape` is only good when the deck is genuinely built around cheap cards.

### 3. Power Defect

This shell is excellent when energy and draw are already stable.

Core enablers:

- `Defragment`
- `Machine Learning`
- `Storm`
- `Subroutine`
- `White Noise`

Best payoffs:

- `Echo Form`
- `Creative AI`
- `Signal Boost`
- `Synthesis`
- `Storm`
- `Subroutine`

Selection rule:

- Do not force `Echo Form` or `Creative AI` without energy and frontload.
- `Signal Boost` gets much better once the deck has several premium powers worth duplicating.

### 4. Status Conversion

This is the most STS2-specific Defect package.

Core enablers:

- `Boost Away`
- `Fight Through`
- `Overclock`
- `TURBO`
- `Gunk Up`

Best payoffs:

- `Smokestack`
- `Trash to Treasure`
- `Iteration`
- `Flak Cannon`
- `Rocket Punch`
- `Compact`

Selection rule:

- This package is real, but only draft the payoff once you already create statuses on purpose.
- `Flak Cannon` is a finisher, not an early pick.
- `Compact` is strong when status clutter is actively costing you turns.

## Remove, Transform, and Upgrade Priorities

### Removes

Default rule:

- Remove `Strike` first.

Exceptions:

- In a fully committed zero-cost deck, clunky non-synergy attacks can become worse than `Defend`.
- Remove curses before basics unless the curse is temporary or your deck can actively convert it through a status or exhaust line.

### Transforms

Default rule:

- Transform `Strike` first.

Exceptions:

- In a stable frost/focus deck, bad leftover attacks are often the best transform targets.

### Best Upgrades

Highest impact:

- `Defragment`
- `Glacier`
- `Echo Form`
- `Loop`
- `All for One`
- `White Noise`
- `Skim`
- `Biased Cognition`

Very strong when relevant:

- `Coolheaded`
- `Beam Cell`
- `Charge Battery`
- `TURBO`
- `Flak Cannon`
- `Shatter`
- `Multi-Cast`

Low-value upgrades unless they fix a breakpoint:

- `Strike`
- `Defend`
- `Chaos`
- `Leap`

## Defect Relic Priorities

Top class relics:

- `Data Disk`
- `Runic Capacitor`
- `Gold-Plated Cables`

Very strong and frequently enabling:

- `Power Cell`
- `Symbiotic Virus`
- `Emotion Chip`
- `Metronome`

Notes:

- `Power Cell` is the main reason not to underrate the cheap-card shell.
- `Symbiotic Virus` makes dark-orb lines much easier to justify.
- `Cracked Core` is still strong baseline frontload, so do not overpay to replace it with a merely cute synergy relic.

## Event Choices That Matter Most

### Usually Excellent

- `Dense Vegetation`: remove a card for `11 HP` when safe. Defect benefits a lot from trimming basic attacks.
- `Doors of Light and Dark`: remove `1` over random upgrading `2` unless the deck is already unusually polished.
- `Morphic Grove`: transform `2` for `100 gold` when early basics are still cluttering orb turns.
- `Spirit Grafter`: remove a card over `Metamorphosis` unless you are already very attack-dense and want free hits.
- `Zen Weaver`: premium remove event.

### Strong With Defect-Specific Context

- `Amalgamator`: `Ultimate Defend` is the default pick because Defect usually values stable defense more than another generic attack.
- `Tea Master`: the temporary starting-hand upgrade is very good when your deck hinges on early powers like `Defragment`, `Loop`, `Machine Learning`, or `Storm`.
- `Wood Carvings`: `Toric Toughness` is generally better than `Peck` for Defect because the class values frontloaded block across multiple turns.
- `The Future of Potions?`: upgraded common skill can be excellent because Defect commons include several genuinely premium skills.
- `Luminous Choir`: the `Spore Mind` line is only tolerable if your deck already handles dead cards well. Otherwise the relic line is safer.

### Usually Avoid

- `Field of Man-Sized Holes`: avoid the `Normality` line almost always. Defect often chains multiple cheap cards and orb triggers in one turn.
- `This or That?`: gold is usually cleaner than a random relic plus `Clumsy`.
- `Wellspring`: random potion is usually safer than taking `Guilty` for one remove.

## Drafting Rules That Hold Up

- Do not mix every engine just because the class can technically support them. Pick one primary plan and one backup plan.
- Prioritize orb reliability first. Even the flashy STS2 packages are better when the deck already blocks and cycles well.
- `Data Disk`, `Defragment`, and `Glacier` solve more problems than most rare attacks do.
- If a deterministic remove or transform is competing against an enchant line with unclear output, prefer the deterministic line.
