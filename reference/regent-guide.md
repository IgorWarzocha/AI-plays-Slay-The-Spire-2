# Regent Decision Guide for Current STS2 Data

This guide is based on `runtime/reference-library.json`. It is specific to the current STS2 Regent pool: stars, forge, Colorless creation, Sovereign Blade support, and generated Minion cards.

## Default Draft Plan

If no build is open yet, draft in this order:

1. Reliable `star` generation.
2. Card draw and smoothing.
3. One scaling package.
4. Block that keeps pace with expensive turns.
5. Removes and transforms.

In practice, that means early rewards should usually prefer cards like `Venerate`, `Glow`, `Gather Light`, `Glimmer`, `Photon Cut`, `Know Thy Place`, `Falling Star`, `Manifest Authority`, `Refine Blade`, and `Bulwark` over expensive payoff cards with no economy behind them.

## Best Packages

### 1. Star Economy

This is the default Regent lane and the cleanest source of consistency.

Core enablers:

- `Venerate`
- `Glow`
- `Gather Light`
- `Solar Strike`
- `Hidden Cache`

Best payoffs:

- `Black Hole`
- `Child of the Stars`
- `Radiate`
- `Mini Regent`
- `Galactic Dust`
- `Genesis`
- `The Sealed Throne`

Selection rule:

- If the deck already gains `star` reliably, payoffs that care about spending it become premium.
- `The Sealed Throne` is a signal card because it turns normal Regent turns into economy engines.
- `Mini Regent` and `Galactic Dust` are both much better when the deck actually spends `star` often.

### 2. Forge / Sovereign Blade

This is the most class-defining package after basic star economy.

Core enablers:

- `Refine Blade`
- `Spoils of Battle`
- `Bulwark`
- `Furnace`
- `Summon Forth`

Best payoffs:

- `Conqueror`
- `Parry`
- `Seeking Edge`
- `Sword Sage`
- `The Smith`
- `Fencing Manual`

Selection rule:

- Do not draft too many `Sovereign Blade` payoffs before you can actually produce and improve the blade consistently.
- `Fencing Manual` is excellent because it gives immediate forge without consuming a pick.
- `Seeking Edge` is one of the clearest build-around powers in the class.

### 3. Colorless Creation

This is the main secondary engine in the current pool.

Core enablers:

- `Manifest Authority`
- `Quasar`
- `Bundle of Joy`
- `Spectrum Shift`
- `Orange Dough`

Best payoffs:

- `Arsenal`
- `Regalite`
- `Pillar of Creation`
- `Supermassive`
- `Heirloom Hammer`

Selection rule:

- The shell is strong when Colorless generation is dense. It is weak when the deck only creates cards occasionally.
- `Regalite` is especially clean because it turns ordinary creation into defense.

### 4. Minion Conversion Tempo

This is a real package because the created Minion cards are efficient.

Core enablers:

- `BEGONE!`
- `CHARGE!!`
- `GUARDS!!!`

Best payoffs:

- `Vitruvian Minion`
- `Pillar of Creation`
- `Make It So`
- `Arsenal`

Selection rule:

- `Minion Strike` and `Minion Sacrifice` are both efficient enough that transform effects creating them are not filler.
- `Vitruvian Minion` is a major payoff relic and a reason to lean harder into these cards if you find it.

## Remove, Transform, and Upgrade Priorities

### Removes

Default rule:

- Remove `Strike` first.

Exceptions:

- In a mature forge/colorless deck, leftover low-impact non-synergy attacks can be worse than `Defend`.
- Remove curses before basics unless the curse is temporary and the deck is already very stable.

### Transforms

Default rule:

- Transform `Strike` first.

Exceptions:

- If the deck is already dense with economy and weak on defense, a bad attack may be a better transform target than `Defend`.

### Best Upgrades

Highest impact:

- `Glow`
- `Gather Light`
- `Glimmer`
- `Refine Blade`
- `Furnace`
- `Genesis`
- `The Sealed Throne`
- `Seeking Edge`

Very strong when relevant:

- `Manifest Authority`
- `Bulwark`
- `Know Thy Place`
- `Shining Strike`
- `Convergence`
- `Black Hole`
- `Supermassive`

Low-value upgrades unless they fix a breakpoint:

- `Strike`
- `Defend`
- `Cloak of Stars`
- `Solar Strike`

## Regent Relic Priorities

Top class relics:

- `Fencing Manual`
- `Mini Regent`
- `Galactic Dust`

Very strong and frequently enabling:

- `Regalite`
- `Lunar Pastry`
- `Orange Dough`
- `Vitruvian Minion`

Notes:

- `Divine Right` is already excellent baseline because Regent wants early `star`.
- `Mini Regent` is much better than it looks because it rewards doing what the class already wants to do.
- `Vitruvian Minion` is a real build-shaping relic if you already create Minion cards.

## Event Choices That Matter Most

### Usually Excellent

- `Dense Vegetation`: remove a card for `11 HP` when safe. Regent benefits heavily from cutting filler because so many turns revolve around economy sequencing.
- `Doors of Light and Dark`: remove `1` over random upgrading `2` unless the deck is already extremely clean.
- `Morphic Grove`: transform `2` for `100 gold` when early basics are still cluttering star turns.
- `Zen Weaver`: premium remove event.
- `Whispering Hollow`: transforming a card for `9 HP` is usually better than two random potions.

### Strong With Regent-Specific Context

- `Tea Master`: the temporary starting-hand upgrade is very good when your deck relies on opening with `star` generation or forge setup.
- `Hungry for Mushrooms`: `Fragrant Mushroom` is usually better than `Big Mushroom`. Regent especially dislikes losing `2` opening draws every combat because its setup turns are resource-sensitive.
- `Wood Carvings`: `Toric Toughness` is the better general option for Regent because it supports slower setup turns; `Peck` is mostly for already aggressive decks.
- `Amalgamator`: `Ultimate Defend` is usually better than `Ultimate Strike` because Regent already has enough ways to scale damage but still appreciates efficient defense.
- `Luminous Choir`: the `Spore Mind` line is usually not attractive unless the deck is already very clean and can tolerate a dead draw.

### Usually Avoid

- `Field of Man-Sized Holes`: avoid the `Normality` line almost always. Regent often needs long sequencing turns.
- `This or That?`: the gold is usually cleaner than a random relic plus `Clumsy`.
- `Wellspring`: one remove is usually not worth adding `Guilty`.

## Drafting Rules That Hold Up

- Regent is strongest when `star` generation is stable before the deck starts chasing expensive payoffs.
- Do not mix forge, Colorless creation, and Minion conversion equally. One of them should be clearly primary.
- `The Sealed Throne` and `Genesis` are payoff cards, not excuses to ignore the early curve.
- If a deterministic remove or transform is competing against an enchant line with unclear output, prefer the deterministic line.
