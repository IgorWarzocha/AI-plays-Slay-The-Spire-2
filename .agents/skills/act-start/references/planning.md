# Planning

Use this note when an act starts or when route planning resets.

## Core Rule

- Do not evaluate one node at a time.
- Evaluate the route your deck is actually prepared to survive.
- Plan in chunks, not clicks.

## Act-Start Questions

At the start of an act, answer these before committing to a route:

1. What is the deck already good at?
2. What still kills it?
3. Which upcoming fight is the real threshold check?
4. How many elites has the deck earned right now?
5. What reward, shop, or campfire would actually change pathing permission?

## Route Heuristics

- A route earns elites.
- HP alone does not mean the route is good.
- Gold alone does not mean the route is good.
- One or two decent pickups do not mean the route is good.
- Boss preparation starts immediately, not three floors before the boss.

Before taking an elite path, check:

- does the deck have enough frontload
- does the deck have enough block to survive the bad turns
- does the deck have any real scaling
- does the route offer recovery after the elite
- am I taking this elite because it is correct, or because it is tempting

If those answers are weak, the route has not earned the elite yet.

## Common Failure Mode

A repeated losing pattern is:

- take a short-term value line
- then immediately choose a high-risk path that contradicts that value line

Example:

- take gold because the deck wants safer immediate value
- then route into an elite anyway

That is usually greed disguised as confidence.

## Planning Standard

- Prefer routes that preserve multiple outs.
- Ask what the path looks like after the next hard room, not just at the next node.
- If a route needs multiple things to go right in sequence, it probably is not the route the deck has earned.
- If a route already forces a hard fight, spend current HP, gold, and potion capacity to improve that fight instead of preserving fake safety into it.
- A route with strong relics, upgraded control, spare potion capacity, and a fire before the boss often has earned one more elite even at imperfect HP.

## Living Rule

This note is meant to evolve. When a repeated pathing mistake or act-planning lesson appears, rewrite this file to absorb it cleanly instead of adding another tiny note.
