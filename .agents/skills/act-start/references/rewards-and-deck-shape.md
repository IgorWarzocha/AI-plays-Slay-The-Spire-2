# Rewards And Deck Shape

Use this note when evaluating card rewards, potions, relics, removes, and deck direction at act level.

## Core Rule

A reward is good if it solves the next real problem the run will face.

Do not take a card because it is generically strong. Take it because it improves:

- weak opening turns
- survival on bad draws
- draw or energy consistency
- AoE coverage
- scaling for the act boss or next elite
- deck compactness and hand quality

## Deck-Shape Checks

When reconstructing the deck, answer:

- Is this mostly a frontload deck, a scaling deck, or an in-between deck that still needs one more piece?
- How many weak or medium cards can it still afford to carry?
- Is draw quality falling because of filler, statuses, or too many cute payoffs?
- Is the deck missing one critical role, or is it merely uneven?

## Reward Heuristics

### Card rewards

- Respect skip. Preserving future draw quality is often better than taking a medium card.
- Prioritize cards that solve immediate failures before speculative payoff cards.
- Do not over-draft synergy. Payoff is earned only after the shell exists.
- If a card is only good after two more hits, it is usually not the pick unless the current run is already stable.

### Potions

- A potion slot is part of current combat power, not future decoration.
- Take potions that change elite, boss, or recovery math.
- Skip or replace low-impact potions when the slot is needed for a real breakpoint potion.

### Relics

- Judge relics by what they change right now:
  - setup quality
  - draw consistency
  - damage conversion
  - block conversion
  - shop or route permission
- Immediate value should change pathing only when it genuinely changes threshold checks.

### Removes and transforms

- Deterministic deck quality is often better than opaque variance.
- When a remove or transform clearly sharpens the next few floors, take it seriously.
- A cleaner deck can be more valuable than one more medium reward.

## Failure Modes To Avoid

- drafting for a future shell that the current run has not opened
- taking immediate value, then pathing like a high-roll deck anyway
- using rewards to widen the deck instead of sharpening the plan
- valuing unknown or weakly surfaced outputs above concrete upgrades to deck quality
