# Bygone Effigy

## What It Is

- Act 1 elite
- Closest useful STS1 analogue:
  - `Lagavulin` as a wake-up threshold fight

## Current Read

- This is not a normal face race.
- The key danger starts after the wake-up threshold and turns into repeated heavy pressure.
- The real readiness question is whether the deck can either end the fight quickly once the pattern starts or survive those heavy turns cleanly.

## Practical Rule

- Do not path into this fight just because current HP looks comfortable.
- Ask a stricter elite-readiness question:
  - can this deck kill fast enough after the enemy's real pattern starts?

## Failure Mode To Avoid

- taking a short-term value line, then immediately forcing this elite anyway
- treating the fight like a hallway damage race instead of a threshold fight
