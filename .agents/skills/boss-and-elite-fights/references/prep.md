# Prep

Use this note before committing to elites or bosses and when checking whether the run has earned a hard fight.

## Core Rule

Hard fights are threshold checks, not generic hallway fights with bigger numbers.

Do not commit because current HP looks fine. Commit because the deck has earned the fight.

## What To Check Before An Elite

- frontload
- survivability through weak opening draws
- scaling or a clear way to end the fight before scaling matters
- draw consistency
- potion quality and empty slots
- whether the route offers recovery after the fight

If those checks are weak, the elite is usually not earned yet.

## Route Earns Elite Logic

A route has earned an elite when most of the following are true:

- the deck can survive its weakest realistic opening draw
- there is a concrete plan for statuses, adds, or phase transitions
- current relics or potions materially improve the matchup
- the post-fight path still has outs
- the relic reward is worth more than the safer line's win percentage

## Boss Prep Over The Last Floors

Boss preparation is not a floor-16 problem.

As the boss approaches, ask:

- what does this boss actually test
- what is still missing in the deck
- should the next reward, fire, or shop be judged almost entirely through the boss
- is the run preserving potions for the turns that matter
- if the boss offers a tax choice, which tax this specific deck can really carry
- whether the deck's draw is passive or energy-paid, and whether hand size itself contributes to defense or retention value

Late in an act, do not over-preserve if the deck is already ahead of the act's tests and the route still has recovery or potion outs.

## STS1 Parallels

STS1 analogies are useful only when they clarify the threat model.

Use them to ask:

- is this really a race fight
- is this a setup fight
- is this a board-control fight
- is this a breakpoint fight
- what is the real clock or loss condition

Do not assume STS2 fights share the same punish windows just because the analogy feels familiar.

## Potion Rule

Potions are for breakpoint turns.

Spend them when they change:

- setup permission
- lethal math
- survival through the next cycle
- whether a body leaves the board this turn
