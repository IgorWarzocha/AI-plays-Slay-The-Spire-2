# Card And Relic Priorities

Use this note for reward, remove, upgrade, relic, and event decisions in Ironclad runs.

## High-Priority Cards By Role

Usually happy to take early:

- `Thunderclap`
- `Uppercut`
- `Shrug It Off`
- `Burning Pact`
- `Headbutt`
- `Battle Trance`
- `Bloodletting`
- `Flame Barrier`
- `Whirlwind`
- `Offering`

Powerful, but only with support:

- `Barricade`
- `Juggernaut`
- `Body Slam`
- `Perfected Strike`
- `Hellraiser`
- `Rupture`
- `Pact's End`
- `Corruption`
- `Stoke`
- `Tear Asunder`

Premium rares and engines to reconsider even when they are not the current deck's obvious card:

- `Dark Embrace`
- `Impervious`
- `Fiend Fire`
- `Corruption`
- `Offering`

Rule:

- Do not treat these as automatic picks, but do force an explicit compare against the "obvious" reward.
- Ask whether the card is merely off-plan now, or whether it would actually create a better plan over the next few floors.
- This matters most on rare rewards, where the highest-upside card is often not the one that best matches the deck's current surface description.
- If you already have one or more exhaust enablers, `Dark Embrace` specifically deserves a stronger compare than most premium rares because it converts those cards from support into a real engine.
- The main correction is about the rare itself, not a blanket downgrade to later `Burning Pact`s. Because `Dark Embrace` is rare here, the first offered copy must be taken more seriously than old STS1 instinct suggests.

Usually lower priority in solo unless the reason is concrete:

- `Demonic Shield`
- `Tank`
- `Colossus`
- `Primal Force`
- `Cascade`
- `Stampede`

## Remove, Transform, And Upgrade Priorities

### Removes

Default rule:

- Remove `Strike` first.

Exceptions:

- Remove `Defend` first if committed to `Perfected Strike` or `Hellraiser`.
- Remove curses before basics unless the curse is temporary or exhaust-friendly and the basic card is actively ruining the deck.

### Transforms

Default rule:

- Transform `Strike` first.

Exceptions:

- Transform `Defend` first in the strike package.
- In a strong block shell with `Body Slam` or `Unmovable`, bad attacks can be better transform targets than `Defend`.

### Best upgrades

Highest impact:

- `Offering`
- `Dark Embrace`
- `Barricade`
- `Body Slam`
- `Pyre`
- `Infernal Blade`
- `Havoc`
- `Stampede`
- `Battle Trance`
- `Bloodletting`

Very strong when relevant:

- `Whirlwind`
- `Fiend Fire`
- `Flame Barrier`
- `Tremble`
- `Thunderclap`
- `Uppercut`
- `Rupture`

Low-value upgrades unless they fix a breakpoint:

- `Strike`
- `Defend`
- `Anger`
- `Iron Wave`

## Ironclad Relic Priorities

Top class relics:

- `Paper Phrog`
- `Ruined Helmet`
- `Charon's Ashes`

Very strong and frequently enabling:

- `Demon Tongue`
- `Self-Forming Clay`
- `Red Skull`

Situational:

- `Brimstone`

Notes:

- `Brimstone` is only correct when the deck can kill quickly or heavily outscale the enemy strength buff. It is much better in bursty vulnerable decks than in slow control decks.
- `Burning Blood` stays excellent because so many strong Ironclad lines spend HP for tempo or scaling.

## Event Choices That Matter Most

These are the decisions with the clearest Ironclad bias in the current event set.

### Usually excellent

- `Dense Vegetation`: remove a card for `11 HP` if life total is stable. Ironclad can repay HP better than most classes.
- `Doors of Light and Dark`: remove `1` over random upgrading `2` unless the deck is already clean and has many premium upgrade hits.
- `Morphic Grove`: transform `2` for `100 gold` when Act 1 basics are still clogging the deck.
- `Spirit Grafter`: remove a card over `Metamorphosis` unless the deck specifically wants random free attacks for attack-count synergies.
- `Whispering Hollow`: transforming a card for `9 HP` is usually better than two random potions.
- `Zen Weaver`: pay for removes whenever the economy allows it. This is one of the cleanest Ironclad improvement events.

### Strong with Ironclad-specific context

- `Amalgamator`: take `Ultimate Strike` if the run is on strike, strength, or offense; take `Ultimate Defend` if the deck is block-focused or still dies to tempo loss.
- `Hungry for Mushrooms`: `Fragrant Mushroom` is usually better than `Big Mushroom`. Losing `2` opening draws every combat is a real cost; Ironclad usually prefers the random upgrades.
- `Luminous Choir`: remove `2` plus `Spore Mind` is acceptable only if the deck already has meaningful exhaust. Otherwise the `149 gold` relic option is safer.
- `Grave of the Forgotten`: `Forgotten Soul` is the safe default. The `Decay` plus `Soul's Power` line is only worth it when exhaust is already core and the target card is excellent.
- `Wood Carvings`: `Toric Toughness` is the better general pickup; `Peck` is better only when the deck already cares about strength or repeated-hit attacks.
- `Tea Master`: buy the temporary strength buff only when important elites or the boss are next. Otherwise the cheaper opening-hand upgrade effect is the better short-term spend.

### Usually avoid

- `Field of Man-Sized Holes`: avoid the `Normality` line almost always. Ironclad has explosive turns and hates the three-card cap.
- `This or That?`: prefer the gold over a random relic plus `Clumsy` unless the deck is already happy to exhaust dead cards.
- `Wellspring`: random potion is usually safer than taking `Guilty` for one remove.
- `Unrest Site`: full heal plus `Poor Sleep` is risky because `Poor Sleep` retains and clogs future turns. Take it only if survival is otherwise in doubt.

## Living Rule

This note should evolve with live runs. Rewrite priority lists when the pool proves them wrong instead of letting stale rankings linger beside new ones.
