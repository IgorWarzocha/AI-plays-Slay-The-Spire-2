# Core Plan

Use this as the main character-level planning note for Ironclad runs.

## What Is Different From STS1

- `Paper Phrog` is much stronger because this pool has a deep `Vulnerable` package: `Bash`, `Break`, `Tremble`, `Thunderclap`, `Uppercut`, `Taunt`, `Molten Fist`, `Dismantle`, `Bully`, `Dominate`, `Vicious`, and `Cruelty` all care directly about `Vulnerable`.
- Ironclad has a real self-damage package now. `Bloodletting`, `Hemokinesis`, `Breakthrough`, `Brand`, `Offering`, `Inferno`, `Spite`, `Rupture`, `Tear Asunder`, `Demon Tongue`, `Self-Forming Clay`, and `Red Skull` all point in that direction.
- Exhaust is deeper than old Ironclad. `Ashen Strike`, `Forgotten Ritual`, `Pact's End`, `Stoke`, and `Howl from Beyond` give more reasons to convert exhaust from cleanup into a main scaling plan.
- Some cards are not STS1-equivalent role players. `Demonic Shield` and `Tank` are ally-facing cards, so they are much lower priority in solo runs. `Corruption` is an `Ancient` build-around, not something to force blindly.

## Default Draft Plan

If no clear build is open yet, draft in this order:

1. reliable `Vulnerable`
2. card flow and energy
3. premium scaling package
4. AoE
5. deck-thinning events and removals

In practice, that means early rewards should usually prefer cards like `Thunderclap`, `Tremble`, `Uppercut`, `Shrug It Off`, `Pommel Strike`, `Headbutt`, `Burning Pact`, `Bloodletting`, `Battle Trance`, `Offering`, `Flame Barrier`, and `Whirlwind` over slow payoff cards with no shell.

## Best Packages

### Vulnerable burst

This is the most reliable default Ironclad lane in the current pool.

Core enablers:

- `Thunderclap`
- `Tremble`
- `Uppercut`
- `Bash`
- `Taunt`

Best payoffs:

- `Paper Phrog`
- `Cruelty`
- `Dismantle`
- `Bully`
- `Dominate`
- `Molten Fist`
- `Break`

Selection rule:

- If the deck already has `2` strong `Vulnerable` applications, aggressively prioritize payoff attacks and relics over more setup.
- `Paper Phrog` is a premium pickup and often reason enough to lean harder into this lane immediately.
- `Molten Fist` is much better than it looks because doubling `Vulnerable` turns later burst cards on.

### Exhaust value

This is the best scaling shell when the deck starts to bloat or picks up exhaust relic support.

Core engine:

- `Burning Pact`
- `True Grit`
- `Second Wind`
- `Feel No Pain`
- `Dark Embrace`

Best payoffs:

- `Fiend Fire`
- `Ashen Strike`
- `Forgotten Ritual`
- `Pact's End`
- `Stoke`
- `Charon's Ashes`
- `Forgotten Soul`

Selection rule:

- Do not draft payoff first unless it is independently playable. `Fiend Fire` is fine early; `Pact's End` is not.
- Once the deck has both `Feel No Pain` and `Dark Embrace`, exhaust cards become premium, not filler.
- `Corruption` is strongest when the deck already has real skill density plus one of `Feel No Pain` or `Dark Embrace`.

### Self-damage strength

This shell is real and much better than old “just Hemokinesis sometimes.”

Core enablers:

- `Rupture`
- `Bloodletting`
- `Hemokinesis`
- `Offering`
- `Brand`
- `Inferno`

Best payoffs:

- `Tear Asunder`
- `Spite`
- `Fight Me!`
- `Red Skull`
- `Self-Forming Clay`
- `Demon Tongue`
- `Ruined Helmet`

Selection rule:

- `Rupture` is a signal card. If the deck already has cheap self-damage, take it aggressively.
- `Demon Tongue` is exceptional here because it turns the first self-hit on your turn into effectively free scaling.
- `Ruined Helmet` sharply raises the value of `Inflame`, `Brand`, `Fight Me!`, `Dominate`, `Setup Strike`, and even the first `Rupture` trigger each combat.

### Block conversion

This package is good when the offered cards push the deck defensive instead of explosive.

Core cards:

- `Shrug It Off`
- `Flame Barrier`
- `True Grit`
- `Impervious`
- `Stone Armor`
- `Unmovable`

Payoffs:

- `Body Slam`
- `Grapple`
- `Juggernaut`
- `Barricade`
- `Toric Toughness`

Selection rule:

- `Barricade` is not an early blind pick. It is great after the deck already has dense block generation or `Unmovable`.
- `Body Slam` becomes premium only after the deck can regularly make `15+` block in one turn.
- `Juggernaut` improves a lot if the deck also contains repeated block sources like `Rage`, `Crimson Mantle`, `Stone Armor`, or `Toric Toughness`.

### Strike package

This package is playable, but only when it is clearly open.

Core cards:

- `Perfected Strike`
- `Pommel Strike`
- `Twin Strike`
- `Setup Strike`
- `Hellraiser`

Best support:

- `Headbutt`
- `One-Two Punch`
- `Ultimate Strike`
- `Paper Phrog`
- strength relics and cards

Selection rule:

- Do not take `Perfected Strike` and then keep removing `Strike` cards out of habit.
- If committed to this lane, remove `Defend` first and keep `Pommel Strike`, `Twin Strike`, `Setup Strike`, `Ashen Strike`, and `Ultimate Strike`.
- `Hellraiser` is strong only when the deck is still dense with cards containing `Strike`.

## Current Live Reminder

Ironclad usually wins by turning raw cards into a coherent damage engine faster than the act punishes dead draws. Starting health is permission to invest selectively, not to draft sloppily.

## Drafting Rules That Hold Up

- Do not force a `3`-cost power deck without energy. `Barricade`, `Demon Form`, `Howl from Beyond`, `Bludgeon`, and `Stomp` are not a plan by themselves.
- Do not over-draft payoff before setup. One `Vulnerable` card plus one payoff card is better than three payoff cards and no setup.
- Ironclad in this pool is rewarded for spending HP deliberately, not randomly. If the HP payment gives deck quality, tempo, or repeatable scaling, it is often correct.
- Exhaust is not just cleanup. When the engine pieces are already present, it becomes one of the strongest late-game plans available.
- If a deterministic remove or transform is competing against an enchant option with unclear output, prefer the deterministic line. The local library does not fully surface enchant effect text, so opaque enchant outcomes should not be over-valued.

## Automation Relevance

Frontload, vulnerable application, draw quality, and setup permission are the recurring things to re-check before acting. If the deck starts to look like a setup deck, stop playing late hallway fights like trivial tempo checks.
