# Heuristics

Use this note for merchant decisions.

## Core Rule

Spend to solve the run's real bottleneck, not to maximize raw quantity.

## Shop Classification

Before buying anything, classify the shop:

- damage-control shop
- survival shop
- forced-elite prep shop
- boss-prep shop
- value shop

That classification should change how aggressively to buy cards, potions, or removes.

## Bottleneck Check

Identify what the deck is actually missing:

- weak opening hand
- no block smoothing
- no scaling
- no AoE
- bloated deck
- empty potion slots before hard fights

## Practical Rules

- Forced hard fight next:
  - value relics that improve setup or consistency immediately
  - removes that sharpen the next shuffle
  - potions if slots are open
  - only buy cards that clearly improve the next fight
- Damage-control shop:
  - assume the route already imposed a tax
  - spend to survive the next few floors, not to preserve gold efficiency
  - fill potion slots aggressively
  - prefer consistency and bailout tools over greedy scaling
- Comfortable shop:
  - prefer one excellent purchase over multiple medium ones

## Remove Vs Buy Vs Hold

- A remove is often the cleanest purchase when the deck is bloated or inconsistent.
- Do not buy filler just because gold is available.
- Holding gold is correct when the current shop does not solve a real problem.

## Potions

- Do not leave a shop with empty potion slots if the next hard fight can justify filling them.
- Potions are part of current combat power and should be valued like temporary relic text.

## Failure Modes To Avoid

- buying raw value when the next few floors are survival checks
- leaving a safety shop without using potion capacity
- forgetting that a shop can enable an elite line or still fail to do so

## Living Rule

Keep this note practical and evolving. Merge repeated shopping lessons into these sections instead of splitting shop logic into many fragments.
